agora.client.account.resolution module¶

class agora.client.account.resolution.AccountResolution(value)[source]¶

Bases: enum.IntEnum

Used to indicate which type of account resolution should be used if a transaction on Kin 4 fails due to an account being unavailable.

EXACT: No account resolution will be used. PREFERRED:

If an account in a request was not found, the client will attempt to resolve the provided account to a Kin token account.

When used for a sender key in a payment or earn request, if Agora is able to resolve the original sender public key to a set of token accounts, the original sender will be used as the owner in the Solana transfer instruction and the first resolved token account will be used as the sender.

When used for a destination key in a payment or earn request, if Agora is able to resolve the destination key to a set of token accounts, the first resolved token account will be used as the destination in the Solana transfer instruction.
